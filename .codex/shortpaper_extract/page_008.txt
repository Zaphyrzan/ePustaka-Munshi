 
ABSTRACT  
The  management  of  library  resources  in  secondary  schools  is  critical  for  fostering  
academic
 
excellence.
 
At
 
SMK
 
Abdullah
 
Munshi,
 
the
 
library
 
currently
 
relies
 
on
 
traditional
 
manual
 
methods,
 
specifically
 
using
 
handwritten
 
physical
 
ledgers
 
to
 
record
 
book
 
inventories
 
and
 
borrowing
 
transactions.
 
This
 
manual
 
approach
 
has
 
led
 
to
 
significant
 
operational
 
inefficiencies,
 
including
 
difficulty
 
in
 
tracking
 
book
 
status,
 
deterioration
 
of
 
physical
 
records,
 
and
 
a
 
time-consuming
 
borrowing
 
process.
 
To
 
address
 
these
 
challenges,
 
this
 
project
 
proposes
 
the
 
development
 
of
 
"ePustaka
 
Munshi:
 
Smart
 
Library
 
System
 
with
 
OCR
 
Ledger
 
Digitization."
 
This
 
web-based
 
application
 
is
 
designed
 
to
 
modernize
 
the
 
library's
 
workflow
 
by
 
integrating
 
Optical
 
Character
 
Recognition
 
(OCR)
 
technology
 
to
 
digitize
 
legacy
 
handwritten
 
records
 
into
 
a
 
centralized
 
database.
 
Developed
 
using
 
the
 
Agile
 
Scrum
 
methodology,
 
the
 
system
 
utilizes
 
React.js
 
for
 
the
 
user
 
interface
 
and
 
Supabase
 
for
 
backend
 
data
 
management.
 
Key
 
features
 
include
 
a
 
specialized
 
OCR
 
verification
 
module
 
for
 
librarians,
 
a
 
simplified
 
barcode
 
scanning
 
interface
 
for
 
library
 
prefects,
 
and
 
a
 
role-based
 
access
 
system
 
to
 
ensure
 
data
 
security.
 
The
 
outcome
 
of
 
this
 
project
 
is
 
a
 
functional,
 
scalable
 
digital
 
platform
 
that
 
preserves
 
historical
 
data
 
while
 
significantly
 
reducing
 
the
 
administrative
 
workload
 
for
 
library
 
staff,
 
ultimately
 
creating
 
a
 
more
 
efficient
 
library
 
environment
 
for
 
the
 
school
 
community.
