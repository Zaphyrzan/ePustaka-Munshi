 
4.2.2  Functional  Requirement  
The  capabilities  and  functionality  of  the  Smart  Library  Management  System  are  
defined
 
in
 
these
 
requirements,
 
which
 
outline
 
how
 
the
 
system
 
can
 
facilitate
 
activities
 
in
 
the
 
SMK
 
Abdullah
 
Munshi
 
library.
 
These
 
requirements
 
are
 
intended
 
to
 
overcome
 
the
 
limitations
 
associated
 
with
 
the
 
current
 
manual
 
system
 
in
 
use
 
while
 
still
 
being
 
operable
 
in
 
the
 
school
 
setting.
 
The
 
functions
 
have
 
been
 
categorized
 
according
 
to
 
the
 
involved
 
users
 
in
 
the
 
system,
 
which
 
include
 
the
 
Librarian,
 
Library
 
Prefect,
 
and
 
Student
 
users.
 
Specific  functions  for  each  user  role  are  provided  based  on  the  nature  of  their  tasks.  
The
 
overall
 
functional
 
needs
 
have
 
been
 
determined
 
by
 
discussions
 
among
 
the
 
stakeholders,
 
observation
 
of
 
the
 
nature
 
of
 
the
 
library
 
process
 
as
 
it
 
exists
 
today,
 
and
 
the
 
objectives
 
of
 
the
 
proposed
 
project
 
as
 
described
 
in
 
Chapter
 
1.
 
 Table  4.1:  Description  of  Module  and  Functions  for  ePustaka  Munshi:   Smart  Library  System  Use  case    Function  Description  UC001  Login  to  System  
This  use  case  allows  authorized  users  (Admin,  Librarian,  Library  Prefect,  Students  and  Teachers)  to  log  in  to  the  system  using  valid  credentials  in  order  to  access  system  functionalities  based  on  their  roles.  UC002  Manage  Student  Accounts  
This  use  case  allows  the  Admin  to  create,  update,  view,  and  deactivate  student  user  accounts  to  control  access  to  the  library  system.  
UC003  Manage  Library  Prefect  Accounts  
This  use  case  allows  the  Admin  to  manage  Library  Prefect  accounts,  including  creating,  updating,  and  assigning  permissions  for  assisting  library  operations.  
UC004  View  Book  Catalog  
This  use  case  allows  users  to  view  the  list  of  available  books  in  the  library,  including  basic  details  such  as  title,  author,  and  availability  status.
